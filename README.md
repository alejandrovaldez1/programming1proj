# Sales Transaction Analysis



## Overview

This project analyzes a retail transaction dataset from Kaggle to answer several business questions about sales trends, popular products, and customer profitability.



## Research Questions

1. How was the sales trend over the months?
2. What are the most frequently purchased products?
3. How many products does the customer typically purchase in each transaction?
4. What are the most profitable customers?
5. Based on the findings, what strategy could the business use to gain more profit?





## Files

* `analysis.ipynb` – main analysis notebook
* `analysis.html` – exported HTML version of notebook
* `data/Sales Transaction v.4a.csv` – dataset
* `requirements.txt` – Python dependencies

## How to Run

1. Open VS Code
2. Install requirements: `pip install -r requirements.txt`
3. Go to the analysis.ipynb file in VS Code.
4. Run all cells from top to bottom.
